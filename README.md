# 项目自述

## 📌 1. 项目简介

`separate.py` 采用分离变量法求解特定微分方程 **dy/dx = x/y** 的数值解。

## ✅ 3. 核心结论

动态区间优化效果显著：

① 与解析解完全吻合（误差为0）；

② 与高精度RK45方法的RMSE降至**0.0045**，稳定性极强。可可靠应用于 **dy/dx = x/y** 的数值求解。

## 💡 4. 使用说明

`separate.py ` 仅支持求解微分方程 **dy/dx = x/y**，运行即可获得准确的数值解。

`variable_separable_solver` 求解形如 dy/dx = f(x) * g(y) 的通用变量分离型一阶常微分方程(数值方法，积分 + 求根，非解析方法)
