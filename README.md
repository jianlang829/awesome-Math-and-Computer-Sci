# 🌐 Discrete & Continuous：一场大二自嗨的数学漫游

> 在教室里发呆，不如在键盘上把公式敲成诗。  
> 这个仓库是我大二“摸鱼”的副产品——把离散数学的图论、微分方程的解析、以及三重积分的“立体”浪漫，打包成一份可翻阅、可复现、可吐槽的学习笔记。  
> 如果你也曾盯着黑板怀疑人生，希望这里能让你重新发现数学的惊喜与温度。

---

## 📦 项目结构一览（只聊思路，不剧透答案）

| 目录 | 关键词 | 你将遇见 |
|---|---|---|
| ` 二元关系 ` | 二元关系 • 邻接矩阵 • 可达性 | 把“人与人认识”翻译成 0 和 1，再让矩阵告诉你谁跟谁真的“连通” |
| ` 图和通路与回路 ` | 通路计数 • 回路判定 • 幂运算 | 原来 Aⁿ 的 (i,j) 位就是在说“从 i 到 j 恰好 n 步”的情书数量 |
| `三重积分` | 柱壳 • 投影 • 换序 | 1≤x≤2, 0≤z≤y≤x 的奇怪区域，如何用积分给它量体积、称重量 |
| `微分方程计算` | 变量分离 • 齐次 • 线性型 | 把“dy/dx = 某个你看不懂的东西”拆成三份套路，让电脑先跑通，再让你跑通 |
| `其他` | 踩坑笔记 • 算法陷阱 • 调试表情包 | 血泪提示：为什么矩阵乘法写反了会算出“幽灵边”？为什么换序积分会突然爆炸？ |

## 🎯 你能带走什么

1. **直觉**  
   图不是点与线，是关系在跳舞；矩阵不是数字块，是关系在说话。  
2. **套路**  
   三重积分先画图再投影，ODE 先判型再换元——模板在手，考试不慌。  
3. **快乐**  
   看终端刷出“Answer: π²/8”时，那种“我算出了宇宙”的中二爽感。  
4. **避坑**  
   附录里每张“踩坑截图”都配了“人话翻译”，让你不再重蹈作者覆辙。
