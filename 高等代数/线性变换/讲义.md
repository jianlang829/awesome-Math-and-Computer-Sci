### 线性变换核心内容
- **向量空间映射**：保持加法和数乘，即 T(a**u**+b**v**)=aT(**u**)+bT(**v**)。
- **矩阵表示**：任何线性变换都能用一个矩阵 **A** 表达，计算就是 **y** = **Ax**。
- **可复合、可逆**：复合=矩阵乘；若 det≠0 则存在逆矩阵，可还原原向量。

### 在计算机里能干什么
1. 图形学：旋转、缩放、平移（用齐次坐标）  
2. 数据降维：PCA 把高维数据投影到低维，保留最大方差  
3. 压缩/滤波：JPEG 的 DCT、卷积核都是线性变换  
4. 机器学习：全连接层、卷积层本质就是线性映射+非线性激活  

### 30 秒可验证的小例子
**问题**：把 2D 点逆时针转 90°  
**矩阵 A** = [[0, -1], [1, 0]]  
**任取向量 x** = [2, 1]ᵀ  
**代码**（Python，无依赖）：
```python
import numpy as np
A = np.array([[0, -1], [1, 0]])
x = np.array([2, 1])
print(A @ x)   # 输出 [-1  2]
```
手绘也能秒算：新坐标 (-1, 2)，与原向量垂直且长度不变，旋转完成。
