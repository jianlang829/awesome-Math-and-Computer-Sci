矩阵的核心内容：  
它是一个数表，行 × 列，支持加、乘、逆、转置等运算。

能解决的计算机问题：

1. 图形变换（旋转、缩放）
2. 线性方程组求解
3. 机器学习特征变换

10 秒可验证的例子（Python 3，无第三方库）：

```python
import math
def rotate(x, y, deg):
    rad = math.radians(deg)
    cos, sin = math.cos(rad), math.sin(rad)
    return cos*x - sin*y, sin*x + cos*y
print(rotate(1, 0, 90))   # → (0.0, 1.0)
```

背后只用了 2×2 旋转矩阵 [[cos,-sin],[sin,cos]]。
